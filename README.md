# aiyshwariya.github.io

Personal website of Aiyshwariya Paulvannan Kanmani, a PhD student in Civil Engineering (AI track) at Texas A&M University working on machine learning for environmental sustainability and wildfire resilience.

Live at https://aiyshwariya.github.io

## About

A single, self-contained static page. All styles, scripts, fonts, and the photo are embedded directly in `index.html`, so there is no build step and no dependencies. The `.nojekyll` file tells GitHub Pages to serve the page as-is.

## Editing

Edit the text in `index.html` directly. Colors and type are defined as CSS variables in the `:root` block near the top of the `<style>` section.
